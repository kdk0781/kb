## 1. 프로젝트 파일 구조

```
kb_faq/
├── index.html          ← HTML 뼈대 (거의 수정 불필요)
├── css/
│   └── style.css       ← 디자인·테마 (색상 변경 시 수정)
└── js/
    ├── data.js         ← ★ 콘텐츠 관리 파일 (FAQ 추가·수정 여기서만)
    └── app.js          ← 렌더링·검색 로직 (기능 변경 시 수정)
```

> **원칙** : 내용(FAQ, 카테고리)은 `data.js`만 수정한다.  
> `app.js`, `index.html`, `style.css`는 기능·디자인 변경이 아니면 건드리지 않는다.

## 2. FAQ 항목 추가하기

### 2-1. 기본 구조

`js/data.js` 의 `FAQ_DATA` 배열 맨 끝(또는 해당 카테고리 블록 안)에 객체 하나를 추가한다.

```js
{
  id  : 'XX000',          // 고유 ID — 명명 규칙은 섹션 6 참고
  cat : '카테고리id',      // CATEGORIES 의 id 값과 일치해야 함
  num : 'Q1',             // 화면에 표시되는 번호 (해당 카테고리 내 순번)
  q   : '질문 텍스트',    // 검색 대상
  a   : `답변 HTML`,      // 작성 규칙은 섹션 4 참고
  tags: [],               // [] | ['NEW'] | ['공통'] | ['NEW','공통']
},
```

### 2-2. 실제 예시

```js
// js/data.js — FAQ_DATA 배열 내 원하는 위치에 추가
{
  id  : 'JE001',
  cat : '전세자금',
  num : 'Q1',
  q   : '전세자금대출 신청 자격 요건은 어떻게 되나요?',
  a   : `<p class="ans-main">무주택 세대주(세대원 포함)로서 아래 요건을 모두 충족해야 합니다.</p>
<ul>
<li>부부합산 연소득 <em>7천만원 이하</em> (신혼가구·2자녀 이상 가구는 8.5천만원)</li>
<li>순자산 기준 충족 (공시가격 합산 3.61억원 이하)</li>
<li>임차보증금 수도권 <strong>3억원 이하</strong>, 지방 <strong>2억원 이하</strong></li>
</ul>
<div class="note">※ 대출 신청일 현재 기준으로 판단하며, 실행일까지 요건 유지 필요</div>`,
  tags: ['NEW'],
},
```

## 3. 새 카테고리 추가하기 — 전세자금대출 예시

카테고리를 새로 만들려면 `js/data.js` 안에서 **3곳**을 순서대로 수정한다.

### Step 1 — `CATEGORIES` 배열에 항목 추가

```js
// js/data.js
const CATEGORIES = [
  { id: 'all',      label: '전체',          icon: '📋' },
  { id: '구입자금',  label: '구입자금',       icon: '🏠' },
  { id: '생활안정자금', label: '생활안정자금',  icon: '💰' },
  { id: '임차보증금', label: '임차보증금 반환', icon: '🔑' },
  { id: '신용대출',  label: '신용대출',       icon: '💳' },
  { id: '공통',     label: '공통',           icon: '📌' },

  // ↓ 새 카테고리 추가
  { id: '전세자금', label: '전세자금대출',    icon: '🏢' },
];
```

> `id` 값은 영문 또는 한글 모두 가능하지만, FAQ 항목의 `cat` 값과 **정확히 일치**해야 한다.

---

### Step 2 — `CAT_COLORS` 객체에 색상 추가

```js
// js/data.js
const CAT_COLORS = {
  '구입자금':    { dark: '#1a3a6b', light: '#e8f0fb' },
  '생활안정자금': { dark: '#2d6a4f', light: '#e8f5ee' },
  '임차보증금':  { dark: '#6b3a1a', light: '#faeee8' },
  '신용대출':    { dark: '#4a1a6b', light: '#ede8fa' },
  '공통':        { dark: '#555',    light: '#f0f0f0' },

  // ↓ 새 카테고리 색상 추가
  // dark  : 그룹 헤더 아이콘 글자색 / 카테고리 대표색
  // light : 그룹 헤더 아이콘 배경색 (dark의 연한 버전 권장)
  '전세자금':    { dark: '#1a5f7a', light: '#e0f3fa' },
};
```

**색상 선택 팁**

| 계열    | dark 예시   | light 예시  |
|---------|-------------|-------------|
| 파랑 계열 | `#1a3a6b` | `#e8f0fb`  |
| 초록 계열 | `#2d6a4f` | `#e8f5ee`  |
| 청록 계열 | `#1a5f7a` | `#e0f3fa`  |
| 갈색 계열 | `#6b3a1a` | `#faeee8`  |
| 보라 계열 | `#4a1a6b` | `#ede8fa`  |
| 회색 계열 | `#555`    | `#f0f0f0`  |

---

### Step 3 — `FAQ_DATA` 배열에 항목 추가

기존 카테고리 블록 아래에 새 블록 주석과 함께 항목들을 추가한다.

```js
// js/data.js — FAQ_DATA 배열
const FAQ_DATA = [

  /* ... 기존 항목들 ... */

  /* ────────────────────────────────────
     전세자금대출                          ← 블록 구분 주석 추가
     ──────────────────────────────────── */
  {
    id  : 'JE001',
    cat : '전세자금',     // Step 1 의 id 와 동일하게
    num : 'Q1',
    q   : '전세자금대출 신청 자격 요건은 어떻게 되나요?',
    a   : `<p class="ans-main">무주택 세대주(세대원 포함)로서 아래 요건을 모두 충족해야 합니다.</p>
<ul>
<li>부부합산 연소득 <em>7천만원 이하</em></li>
<li>임차보증금 수도권 <strong>3억원 이하</strong>, 지방 <strong>2억원 이하</strong></li>
</ul>`,
    tags: ['NEW'],
  },
  {
    id  : 'JE002',
    cat : '전세자금',
    num : 'Q2',
    q   : '전세자금대출 한도는 어떻게 되나요?',
    a   : `<p class="ans-main">임차보증금의 <em>80% 이내</em>에서 최대 2억원까지 가능합니다.</p>
<div class="note">※ 신혼가구·2자녀 이상 가구의 경우 최대 3억원까지 가능</div>`,
    tags: ['NEW'],
  },

];
```

## 4. 답변 HTML 작성 규칙

답변(`a` 필드)은 HTML 문자열이다. 아래 클래스를 활용하면 디자인이 자동 적용된다.

### 4-1. 핵심 답변 강조 — `.ans-main`

```html
<p class="ans-main">가장 중요한 핵심 답변을 여기에 씁니다.</p>
```

> 왼쪽에 골드 보더가 붙는 강조 블록. **항목당 1개만** 사용 권장.

---

### 4-2. 세부 목록 — `<ul><li>`

```html
<ul>
<li>첫 번째 세부 내용</li>
<li>두 번째 세부 내용</li>
</ul>
```

> 자동으로 `○` 불릿과 골드 색상이 적용된다.

---

### 4-3. 주석·참고 박스 — `.note`

```html
<div class="note">※ 참고사항이나 주의사항을 여기에 씁니다.</div>
```

> 회색 배경의 작은 박스. 경고·안내·관련 문서 등에 사용.

---

### 4-4. 인라인 강조 — `<em>`, `<strong>`

```html
<em>골드 색상 강조</em>       ← 핵심 키워드, 중요 수치
<strong>굵게 강조</strong>    ← 제목, 구분자
```

---

### 4-5. 표 — `.faq-table`

```html
<table class="faq-table">
  <thead>
    <tr><th>구분</th><th>내용</th><th>기한</th></tr>
  </thead>
  <tbody>
    <tr><td>항목1</td><td>설명</td><td>3개월</td></tr>
    <tr><td>항목2</td><td>설명</td><td>6개월</td></tr>
  </tbody>
</table>
```

---

### 4-6. 조합 예시 (실무 패턴)

```js
a: `<p class="ans-main">가능합니다.</p>
<ul>
<li>조건 A : <em>필수 충족</em></li>
<li>조건 B : <strong>서류 제출</strong> 후 처리</li>
</ul>
<div class="note">※ 2025.12.31까지만 적용되는 한시 규정입니다.</div>`,
```

## 5. 태그(뱃지) 사용 기준

```js
tags: []              // 기본 — 뱃지 없음
tags: ['NEW']         // 빨간 NEW 뱃지 — 신규 추가 항목
tags: ['공통']        // 네이비 공통 뱃지 — 여러 카테고리에 걸친 규정
tags: ['NEW', '공통'] // 둘 다 표시
```

> **NEW 뱃지 관리 원칙** : 새 FAQ 고시 후 다음 업데이트 시 `'NEW'` 제거.  
> 업데이트 일자를 `header-meta`(`index.html` 6번째 줄)에 반영한다.

## 6. FAQ ID 명명 규칙

| 카테고리      | 접두사 | 예시                        |
|--------------|--------|-----------------------------|
| 구입자금      | `B`    | `B001`, `B002` …           |
| 생활안정자금  | `S`    | `S001`, `S002` …           |
| 임차보증금 반환 | `R`  | `R001`, `R002` …           |
| 신용대출      | `CR`   | `CR001`, `CR002` …         |
| 공통          | `C`    | `C018`, `C019` …           |
| 전세자금대출  | `JE`   | `JE001`, `JE002` …         |
| 새 카테고리   | 영문 약자 2~3자 | 자유롭게 지정     |

> - 번호는 **001부터 3자리** 패딩 권장  
> - 한번 부여된 ID는 변경하지 않는다 (순서 재정렬 시에도 ID 유지)

## 7. 카테고리 색상 커스터마이징

색상을 변경하고 싶을 때는 `js/data.js`의 `CAT_COLORS` 만 수정한다.  
CSS는 건드리지 않아도 된다.

```js
const CAT_COLORS = {
  '전세자금': {
    dark : '#1a5f7a',  // 어두운 색 — 아이콘 글자, 카테고리 대표색
    light: '#e0f3fa',  // 밝은 색  — 아이콘 배경
  },
};
```

**다크 모드에서도 자동 적용**되므로 별도 처리 불필요.

## 8. 자주 하는 실수 & 체크리스트

### ✅ 항목 추가 전 확인

- [ ] `cat` 값이 `CATEGORIES` 배열의 `id`와 **정확히 일치**하는가
- [ ] 새 카테고리라면 `CATEGORIES` + `CAT_COLORS` **둘 다** 추가했는가
- [ ] `num`이 해당 카테고리 내에서 **중복되지 않는가**
- [ ] `id`가 전체 `FAQ_DATA`에서 **유일한가**
- [ ] 답변 HTML 문자열 안에 **백틱(`` ` ``)이 포함되지 않는가** (템플릿 리터럴 종료 오류)

### ⚠️ 흔한 오류

| 증상 | 원인 | 해결 |
|------|------|------|
| 카테고리 탭이 안 보임 | `CATEGORIES` 에 추가 안 함 | Step 1 재확인 |
| 아이콘 색상이 회색 | `CAT_COLORS` 에 없는 `cat` 값 | Step 2 재확인 |
| 화면이 완전히 깨짐 | JS 문법 오류 (쉼표 누락, 따옴표 미닫힘) | 브라우저 콘솔(F12) 확인 |
| 항목이 보이지 않음 | `cat` 값 오타 | `cat` ↔ `CATEGORIES.id` 비교 |

### 📝 업데이트 작업 순서 (권장)

```
1. js/data.js 수정 (카테고리 or FAQ 항목)
2. 브라우저에서 index.html 열어 확인
3. 신규 항목은 tags: ['NEW'] 추가
4. index.html 의 header-meta 날짜 갱신
5. GitHub Pages push
```
